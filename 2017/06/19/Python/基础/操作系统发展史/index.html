操作系统发展史

发表于 2017-06-19

Unix : 一个老头为了玩游戏而写的，后来想让这个系统在多个设备上通用，又开发了 C 语言，然后重写了Unix系统；
Mini-Unix： Unix的源码后来不再开源，一位大学教授写了一个类似于Unix的操作系统，起名字叫 Mini-Unix;
Linux : Torvalds 利用 GNU 的 bash 当做开发环境， gcc 当做编译工具，编写了 Linux 内核 -v0.02;

之后 Linux 又分为了 内核版本 和 发行版本，我们日常中用到的都是发行版本，比如Ubuntu、红帽、CentOs 等等。
