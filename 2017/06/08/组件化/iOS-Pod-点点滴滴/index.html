单元测试

在执行单元测试的时候发现找不到对应的测试文件，导入了之后，提示 not find file xxx.h
两步解决：

  • 修改podfile 文件,测试的target一定要包含进来
  • 测试的 Other Linker flag 和主工程一致

目前来看，这两步设置了之后就可以愉快的测试了。

调试：

在提交私有pod库的时候，并不总是会成功，如果有警告我们可以忽略，但是一旦有error，就不能提交成功了
