XCTest(一) API认识

发表于 2017-05-25

最近要做一些重构的工作，为了让重构更加顺利，决定做一下 iOS 中的单元测试。

OCUnit 是什么？

OCUnit 是 XCTest 的前身，现在已经不怎么使用了。

XCTest 介绍

XCTest 是Xcode集成的测试框架，也就是 XCTest.framework，。通过 XCTest 我们可以在Xcode中创建、执行单元测试和UI测试。

XCTest API

XCTest 的主要类有两个，一个是 XCTestCase，另一个是XCTest。

XCTest 一个测试抽象基类
