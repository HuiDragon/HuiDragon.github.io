发表于 2017-05-08 | 分类于 MySQL

SQL 语言介绍：

SQL，全称数据化结构查询语言，常见的SQL语句类型分为：数据定义语言（DDL）、事务处理语言（TPL）、数据控制语言（DCL）和数据操作语言（DML）。对于开发而言，着重使用的是DML，DML语言包含了Select、Insert、Update和Delete等操作。正确使用SQL语句，有利于：增加数据库处理效率，减少应用相应的时间；减少数据库服务的负载，增加服务器的稳定性；减少服务器间通讯的网络流量。

Join从句

Join分为四种类型：内连接、全外连接、左外连接、右外连接和交叉连接。

内连接（Inner Join）

内连接基于连接为此将两张表（如A表和B表）的列组合在一起，产生新的结果表。
