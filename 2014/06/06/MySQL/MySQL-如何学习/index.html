MySQL 如何学习

发表于 2014-06-06 | 分类于 MySQL

MySQL 已经看了一段时间了，突然觉得应该看看如何去学习了。之前想在一开始要学习的时候就看一下如何学习，但是觉得有些概念或者东西可能不知道，就直接先安装了MySQL，然后写了一些SQL，写着写着就想着如何去提升了。

学习MySQL资料

官方文档

https://dev.mysql.com/doc/

书籍

  1. 《数据库系统概念（原书第六版）》【Abraham Silberschatz（耶鲁大学） Henry F.Korth（利哈伊大学） S.Sudarshan（印度理工学院） 著】[杨冬青 李红燕 唐世渭 (译)] 机械工业出版社
  2. 《深入理解MySQL》
  3. 《深入理解MySQL核心技术》
  4. 《数据库系统实现（第二版）》
  5. 《高性能mysql第三版》

上面罗列了一些学习的书籍，当然还有一些学习博客，这些等着掌握了一定的基础之后再去记录。
