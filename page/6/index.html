MySQL 基本使用

发表于 2014-08-07 | 分类于 MySQL

MySQL 常用命令

  1. 显示当前服务器版本： SELECT VERSION();

  2. 显示当前日期时间： SELECT NOW();

  3. 显示当前用户： SELECT USER();

MySQL 语句规范

  • 关键字与函数名称全部使用大写；
  • 数据库名称、表名称和字段名称全部使用小写；
  • SQL 语句必须以;结尾。
